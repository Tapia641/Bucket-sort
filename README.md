# Bucket-Sort
Ordenamiento de números mediante el uso de hilos en C



### Requerimientos

Compilar en una terminal de _Linux_

Modo de compilación
```
$ g++ BucketSort.cpp -o BucketSort -pthread

```

Uso
```
$ ./BucketSort N

```
Donde N representa el número de cubetas.
